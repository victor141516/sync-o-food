# Sync-o-Food
This Chrome extension will make easy to order food in your office.

## Usage
1. Install the extension
2. Insert your group ID
3. Be (or not) Ryan
4. Take a role:
    1. Be Ryan and order for everyone
    2. Don't be Ryan, choose your food and send it to Ryan

## Supported services
- [x] Deliveroo
- [ ] Uber eats
- [ ] Just-Eat
- [ ] Glovo
